Typography follows the scale defined in the design system. 

**Design reference:** [Figma (Typography)](https://www.figma.com/design/YbvOpVPUBxTSEzg4J8pXKu/Weni---unnnic?node-id=404-210&m=dev)

**Font:** Inter · **Weights:** 400 (Regular), 500 (Medium), 600 (Semibold)

Inter uses **letter-spacing** adjustments for legibility. Each typography token has a matching letter-spacing token (e.g. `$unnnic-font-letter-spacing-display-1`). Use the `font` token for the CSS `font` shorthand and the letter-spacing token for the `letter-spacing` property.

## Usage

Use the `font` token for the CSS `font` shorthand and the matching letter-spacing token for `letter-spacing`:

```scss
.my-heading {
  font: $unnnic-font-display-1;
  letter-spacing: $unnnic-font-letter-spacing-display-1;
}

.my-body {
  font: $unnnic-font-body;
  letter-spacing: $unnnic-font-letter-spacing-body;
}
```
